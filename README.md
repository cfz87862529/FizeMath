# FizeMath
数学计算库。

FizeMath 是一个数学计算库。
FizeMath 对 bc 函数 、 gmp 函数进行面向对象封装。
FizeMath 还提供了常见的一些数学方法，方便调用。
FizeMath 有非常完善的参考文档，其代码结构也经过精雕细琢，精炼而实用。
